# 💰 Smart Wealth Hub

An AI-powered financial analytics and stock prediction web app built using Streamlit. This project helps users analyze stock performance, calculate financial metrics, and get predictive insights using machine learning.

---

## 🚀 Live Demo

https://smart-wealth-pro.streamlit.app/

---

## 📌 Features

* 📊 Stock Price Analysis
* 📈 Data Visualization (Interactive Charts)
* 🤖 Machine Learning-based Stock Prediction
* 💼 CAPM (Capital Asset Pricing Model) Calculations
* 📉 Risk & Return Analysis
* 📂 Category-wise Stock Insights

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Plotly

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/smart-wealth-hub.git
cd smart-wealth-hub
```

### 2️⃣ Create virtual environment (optional)

```
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the app

```
streamlit run app.py
```
